🔴🟢 Squid Game Twitter Analytics Dashboard

An interactive Tableau Dashboard analyzing Twitter activity related to the global hit series Squid Game.

This project focuses on understanding user behavior, tweet patterns, engagement trends, and content distribution during the show's viral period.

📊 Dashboard Overview

The dashboard provides insights into:

👥 Total Users: 54,618

🐦 Total Tweets: 80,019

📝 Tweet Length Distribution

📈 Tweets Over Time (trend analysis)

🌍 User Location & Verification Filters

🏆 Top 10 Users by Followers

📱 Top Tweet Sources (iPhone, Android, Web App, etc.)

📅 Year-wise New User Growth

🎯 Objectives

Analyze social media trends around a viral TV show

Identify tweeting patterns and spikes over time

Discover top influencers discussing the show

Understand which platforms users prefer for tweeting

Build a visually appealing and interactive dashboard

🛠 Tools & Technologies Used

Tableau – Data Visualization & Dashboard Design

Data Cleaning & Preparation

Interactive Filters & KPI Cards

Dark Theme UI for cinematic storytelling

📌 Key Insights

Major tweet spikes during peak popularity periods

Most tweets were posted via Twitter for iPhone

A small number of users hold a very high follower base

Tweet length distribution shows medium-length tweets are most common

🎨 Dashboard Features

Interactive Filters (Source, Sign Up Year, Verified Users, Location)

KPI Summary Cards

Clean Black & Neon Theme for visual impact

Structured layout for easy navigation
